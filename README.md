# Pydoku
## A Sudoku puzzle solver written in Python

### Description
This solver has successfully solved expert level Sudoku puzzles in a couple of seconds.

### Setup
* `python3.8 -m venv ./venv`
* `pip install -r requirements.txt`

### Tests
* `pytest .`
